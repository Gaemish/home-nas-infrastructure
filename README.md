# Home NAS Infrastructure (Raspberry Pi + OMV)

## Overview
Self-hosted NAS providing centralized storage across multiple Linux systems using NFS and SMB.

## Features
- NFS and SMB file sharing
- rsync-based backups
- Static IP + DNS integration
- Multi-device access

## Technologies
- OpenMediaVault
- Raspberry Pi 4B
- Linux (Debian-based)
- NFS / SMB
- rsync

## Lessons Learned
- Troubleshooting network interface failures
- Handling DNS/mDNS conflicts
- Ensuring data integrity during transfers
